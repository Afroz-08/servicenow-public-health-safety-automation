# Public Health Safety Automation – ServiceNow

## Overview
Public Health Safety Automation is a custom scoped ServiceNow application designed to streamline healthcare service requests and appointment workflows through automation and secure lifecycle management.

The application enables citizens to submit healthcare-related requests through a Service Catalog interface while healthcare agents manage approvals, prioritization, appointments, and request tracking.

---

## Features

- Custom Scoped Application
- Service Catalog Integration
- Flow Designer Automation
- Decision Table–Based Priority Assignment
- Automated Approvals & Notifications
- Business Rule Validation
- Role-Based ACL Security
- Appointment Workflow Tracking
- Reports & Dashboards

---

## Technologies Used

## Workflow

Citizen Request Submission  
→ Request Validation  
→ Priority Assignment  
→ Approval Workflow  
→ Appointment Scheduling  
→ Notifications & Status Tracking
